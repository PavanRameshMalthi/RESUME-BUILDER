# Resume Builder

A simple Resume Builder web application that helps users create professional resumes easily.

## Features

- Add personal details
- Add education information
- Add skills and projects
- Real-time resume preview
- Download resume as PDF
- Responsive design
- Dark/Light mode

## Technologies Used

- HTML
- CSS
- JavaScript
- jsPDF

## How It Works
- Enter personal details.
- Add education, skills, projects, and experience.
- View changes instantly in the resume preview.
- Choose a preferred template.
- Toggle between Dark and Light Mode.
- Save progress automatically.
- Export the completed resume as a PDF.

## Project Purpose

This project was built to practice front-end web development concepts such as form handling, DOM manipulation, responsive design, and PDF generation.
